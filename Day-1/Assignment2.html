difference between var , let , const. 

        var :-
        -----
        --var is global scope.
        --It is function scoped.
        --It will not throw an error if use it first and then declare it.
        --you can modifie the value.

        let :-
        -----
        --let is local scoped.
        --It is block scoped.
        --It will throw an error if u use it first and then declare it.
        --you can modifie the value in it.

        const :- 
        -------- 
        --It is also block scoped.
        --It will throw an error if u use it first and then declare it.
        --you can not modifies the value  assigned to it.
